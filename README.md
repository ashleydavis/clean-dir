# clean-dir

A simple command line tool to make a recursively delete node_modules directories.

## Install it

```bash
npm install -g @codecapers/clean-dir
```

## Run it

```bash
clean-dir <directory>
```

## Development

### Run it from code

```bash
cd clean-dir
npm install
npm start
```

### Run it for development

```bash
cd clean-dir
npm install
npm run start:dev
```
